How to Use HG10 Form Builder
🇵🇭 DepEd HG Annex 5 — SF10 DM-OUCI-346-2021 · Legal Size
  1. Header Tab — Set title, grade level, advisory, annex ref, upload logos.
  2. Columns Tab — Fill 3 school-year columns. School Year, School, Grade, Section, Adviser are shared globally (apply to all students). Learning Modality and Parent/Guardian are per-student.
  3. Students Tab — Add one by one or bulk-import via CSV. Each student has their own Learning Modality (×3 columns) and Parent/Guardian (×3 columns).
  4. CSV Format: Name, LRN, LM1, LM2, LM3, Parent1, Parent2, Parent3
  5. Ratings Tab — Select a student and set their scores per competency per column.
  6. Sigs Tab — Set adviser, counselor, school head names for Page 2.
  7. Click any text on the form to edit it live.
  8. Click any score cell to toggle the ✓ checkmark.
  9. Print All — prints every student, 2 pages each on legal size.
⚠️ Set print size to Legal (8.5×14) and margins to Minimum for best results.
