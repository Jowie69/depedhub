How to Use HG9 Form Builder
🇵🇭 Designed for Filipino public school teachers under DepEd.
  1. Header Tab — Fill in school info; auto-populates all forms.
  2. Students Tab — Add students one by one or import via CSV.
  3. Ratings Tab — Select a student, rate their competencies (0–4).
  4. Signatories Tab — Set adviser, counselor, and school head names.
  5. Direct Editing — Click any text in the preview to edit it directly.
  6. Print — Use Ctrl+P. Set paper to A4, no margins.
  7. Export — Export all forms as a printable HTML file.
⚠️ Forms shown 2-per-page as per DepEd HG Annex 5 format.
